# Python: from the basics to the scientific programming

Welcome to the **Python Course** repository! This collection of materials is designed to guide you from the fundamentals of Python programming to advanced topics in scientific computing and data visualization.

## Contents

- **Basics of Python:**  
  Learn Python syntax, variables, data types, and essential data structures (lists, dictionaries, sets, tuples).

- **Object-Oriented Programming:**  
  Understand classes, objects, inheritance, and how to structure code for reusability and clarity.

- **Scientific Libraries & Data Visualization:**  
  Explore popular libraries such as NumPy, pandas, Matplotlib, and Seaborn for data analysis and visualization.

- **Clean Code & Packaging:**  
  Best practices for writing readable, maintainable code and organizing projects for distribution.

## Who is this for?

- Beginners who want to learn Python from scratch.
- Intermediate users looking to deepen their understanding of scientific programming.
- Anyone interested in writing clean, efficient, and well-structured Python code.
